Game

Site-55 is a private RP server in the game SCP:RP on Roblox. We host sessions where you can join and have fun with some of the most respected members of the community!

Discord

The Site-55 Discord server is for all official communications. Here you can see sneak-peeks, major and minor announcements, events and much more!

Departments

Site-55 has several departments, including the Game and Discord Moderation Departments, Morphing Department, Acting Department, Event Hosting Department and the Development Department - Apply in our Discord server!

Site-55 was founded and established by Liam V4 and Co-Founder Vabzie. These users are experienced and recognized within the SCP:RP community. Their team has together built this server and the game map from scratch!
